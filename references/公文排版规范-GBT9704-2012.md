# GB/T 9704—2012《党政机关公文格式》排版规范

> 国家标准全文提取，适用于 DOCX 格式公文排版。凡生成或排版公文文档，必须严格遵守以下规定。

---

## 一、页面设置

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 纸张 | A4 | 210 mm × 297 mm |
| 天头（上白边） | **37 mm ± 1 mm** | 版心上边缘距纸张上边缘 |
| 订口（左白边） | **28 mm ± 1 mm** | 版心左边缘距纸张左边缘 |
| 版心尺寸 | **156 mm × 225 mm** | 成品；右白边 = 210-28-156 = 26 mm，下白边 = 297-37-225 = 35 mm |
| 行数 | **每面 22 行** | 行距以 3 号字高度 + 7/8 3号字高度计 |
| 字数 | **每行 28 个字** | 撑满版心 |
| 文字颜色 | **黑色** | 发文机关标志、分隔线除外 |

> ⚠️ DOCX 排版时：上边距 3.7cm、下边距 3.5cm、左边距 2.8cm、右边距 2.6cm（实际按版心 156×225mm 反算）。

---

## 二、字体与字号

| 要素 | 字体 | 字号 | 颜色 | 说明 |
|------|------|------|------|------|
| **正文** | 仿宋 | **3 号** | 黑 | 默认要素 |
| **标题** | 小标宋体 | **2 号** | 黑 | 居中，可换行 |
| **一级标题（一、）** | **黑体** | 3 号 | 黑 | 单独成行 |
| **二级标题（（一））** | **楷体** | 3 号 | 黑 | 可接正文 |
| **三级标题（1.）** | 仿宋 | 3 号 | 黑 | 可接正文 |
| **四级标题（（1））** | 仿宋 | 3 号 | 黑 | 可接正文 |
| **主送机关** | 仿宋 | 3 号 | 黑 | 顶格 |
| **附件标记（版心左上角）** | **黑体** | 3 号 | 黑 | 顶格 |
| **附件说明** | 仿宋 | 3 号 | 黑 | 左空二字 |
| **发文机关署名** | 仿宋 | 3 号 | 黑 | 右空二字 |
| **成文日期** | 仿宋 | 3 号 | 黑 | 右空四字 |
| **抄送机关** | 仿宋 | **4 号** | 黑 | 左右各空一字 |
| **印发机关和印发日期** | 仿宋 | **4 号** | 黑 | 左/右空一字 |
| **页码** | **半角宋体阿拉伯数字** | **4 号** | 黑 | 左右各一条一字线 |
| **份号** | 阿拉伯数字 | 3 号 | 黑 | 顶格左第一行 |
| **密级和保密期限** | **黑体** | 3 号 | 黑 | 顶格左第二行 |
| **紧急程度** | 黑体 | 3 号 | 黑 | 顶格左 |
| **发文机关标志** | **小标宋体** | — | **红色** | 居中，上35mm |
| **"签发人"** | 仿宋 | 3 号 | 黑 | 居右空一字 |
| **签发人姓名** | **楷体** | 3 号 | 黑 | 居右空一字 |

> 字号对照（参考）：初号=42pt、小初=36pt、一号=26pt、小一=24pt、二号=22pt、**小二=18pt**、三号=16pt、**小三=15pt**、四号=14pt、小四=12pt、**五号=10.5pt**、小五=9pt。
> ⚠️ Word 中的"二号"对应 22pt，"三号"对应 16pt，"四号"对应 14pt。

---

## 三、结构层次与序号

| 层级 | 格式 | 字体要求 | 说明 |
|------|------|----------|------|
| 第一层 | **一、二、三、……** | 3 号黑体 | 单独成行，句末不加标点 |
| 第二层 | **（一）（二）（三）……** | 3 号楷体 | 可单独成行或接正文 |
| 第三层 | **1. 2. 3. ……** | 3 号仿宋 | 接正文 |
| 第四层 | **（1）（2）（3）……** | 3 号仿宋 | 接正文 |

注意：
- 层次序号不得逆向使用（不得没有一、直接写（一））
- 第一层一般用顿号"、"，第二层不加顿号直接用括号
- 第三层用圆点"."（句点+空格）

---

## 四、段落与间距

| 要素 | 规则 |
|------|------|
| 正文缩进 | **每个自然段左空二字**（即首行缩进 2 字符） |
| 标题位置 | 红色分隔线下**空二行**，居中，回行梯形或菱形 |
| 主送机关 | 标题下**空一行**，居左顶格，回行顶格 |
| 正文开始 | 主送机关名称**下一行** |
| 附件说明 | 正文下**空一行**左空二字 |
| 附件标记 | 版心左上角第一行顶格 |
| 发文机关署名 | 正文/附件说明下**空一行**右空二字 |
| 成文日期 | 署名**下一行**右空四字 |
| 版记分隔线 | 首/末条粗线（0.35mm），中间细线（0.25mm） |
| 行距 | 以能排满 22 行为准（固定值约 28.95pt） |
| 字距 | 以能排满 28 字为准（标准字距） |

---

## 五、页眉页脚与页码

| 项目 | 规则 |
|------|------|
| 页码字体 | **4 号半角宋体阿拉伯数字** |
| 页码位置 | 版心下边缘之下，一字线距版心下边缘 **7 mm** |
| 单页码 | 居**右**空一字 |
| 双页码 | 居**左**空一字 |
| 空白页/版记页 | **不编排页码** |
| 附件页码 | 与正文**连续编排** |
| 首页页码 | **正常编排**（信函格式首页不显示） |

---

## 六、版头（红头）要素

| 要素 | 规则 |
|------|------|
| 份号 | 6 位 3 号阿拉伯数字，顶格左第一行 |
| 密级+保密期限 | 3 号黑体，顶格左第二行 |
| 紧急程度 | 3 号黑体，顶格左；三者按份号→密级→紧急程度自上而下 |
| 发文机关标志 | 小标宋体，**红色**，上边缘距版心上边缘 35 mm |
| 发文字号 | 发文机关标志下**空二行**居中；年份用六角括号〔〕；上行文居左空一字 |
| 签发人 | "签发人"3 号仿宋，姓名 3 号楷体，居右空一字 |
| 红色分隔线 | 发文字号下 4 mm，与版心等宽，红色 |

---

## 七、装订与印刷

- **装订方式**：左侧装订
- **印刷方式**：双面印刷
- **页码套正误差**：不超过 2 mm
- **裁切误差**：±2 mm
- **订位**：两钉外订眼距版面上下边缘各 70 mm（±4 mm）


## 九、特殊格式

### 信函格式

- 发文机关标志上边缘至上页边 **30 mm**
- 发文机关标志下 4 mm 处印红色双线（上粗下细）
- 距下页边 20 mm 处印红色双线（上细下粗）
- 线长 170 mm，居中
- 首页**不显示页码**

### 纪要格式

- 纪要标志："XXXXX纪要"，上边缘至版心上边缘 35 mm，红色小标宋
- 出席人员名单：3 号黑体"出席"冒号后 3 号仿宋
