# 风格规则

> 检察机关公文风格规范。目标是：规范、准确、沉稳、自然，避免 AI 腔、宣传腔、夸张表述。

> 📎 另有「用户提供版」（`风格规则-用户提供版.md`）基于本部门实际语料提炼，冲突时优先参考。安全/合规规则（R1-R9）为硬性约束，不可覆盖。

---

## 一、句式规范

### ✅ 推荐

| 类型 | 示例 |
|------|------|
| 主谓宾短句 | "本院依法对某某案提起公诉。" |
| "对……进行……"结构 | "对侦查活动合法性进行监督。" |
| "围绕……开展工作" | "围绕优化营商环境开展专项监督。" |
| 客观陈述 | "2025年1月至6月，共受理审查逮捕案件XX件XX人。" |
| 以事实为主语 | "经审查，某某行为涉嫌构成某罪。" |

### ❌ 不推荐

| 类型 | 示例 | 问题 |
|------|------|------|
| 评价性开头 | "在院党组的坚强领导下" | 套话，除非会议讲话稿需要 |
| 过度排比 | "我们以更高站位、更宽视野、更大力度……" | 空洞 |
| 拟人化 | "法律利剑高高举起" | 文学化，不正式 |
| 感叹式 | "这是多么深刻的教训！" | 不客观 |
| 设问式 | "如何才能防止此类事件再次发生？答案是……" | 非检察机关公文风格 |

## 二、用词规范

### ✅ 推荐用词

| 场景 | 推荐用词 |
|------|----------|
| 表示时间 | "及时""迅速""立即"（不用"第一时间"）|
| 表示数量 | 精确数字 或 【待补充】（不用"大量""众多"）|
| 表示程度 | "较为""相对"（少用"非常""极其"）|
| 表示因果 | "因此""据此""鉴于此"（不用"所以说"）|
| 表示转折 | "但""然而"（不用"可是"）|
| 表示开始 | "启动""开展""组织实施" |
| 表示完成 | "已完成""办结""办结率XX%" |
| 表示取得 | "实现""达成"（不用"斩获""喜提"）|

### ❌ 避免词汇

- 网络流行语：赋能（特定语境可）、抓手（慎用）、闭环（慎用）、底层逻辑
- 宣传话语：成效显著、历史性突破、圆满成功、胜利闭幕、举世瞩目
- 夸张修饰：狠抓、强推、铁腕、零容忍（特定语境可用，但慎用）
- 不确定性弱化语：应该、大概、可能、似乎（在事实陈述中）
- AI 腔：值得关注的是、值得注意的是、毋庸置疑、值得一提的是

## 三、段落规范

1. **首句立题**：每段第一句是中心句，表明本段讲什么
2. **一事一段**：不同事项分不同段落，不混在一起
3. **逻辑递进**：短段在前（概括）、长段在后（展开）
4. **段落长度**：一般 5-8 行，不宜超过 15 行
5. **层次标记**：正确使用一、（一）1.（1）四级标题

---

## 四、语气控制

| 文种 | 语气 |
|------|------|
| 工作汇报 | 客观陈述，适当体现工作主动性，不自我拔高 |
| 请示 | 谦和，突出必要性、可行性 |
| 报告 | 严谨、全面，突出规范性 |
| 情况说明 | 中性、准确、不偏不倚 |
| 会议纪要 | 客观记录，无个人色彩 |
| 信息简报 | 简洁明快，亮点前置 |
| 领导讲话稿 | 庄重、有指导性、有带动性 |
| 案件分析 | 审慎、严谨，区分"事实"与"意见" |

---

## 五、数字与标点

- 数字：一般用阿拉伯数字（"2025年""3项措施"）
- 百分比：用"%"符号（"同比增长15%"）
- 金额：统一单位（"万元"或"亿元"）
- 标点：使用中文全角标点（，。；：""——）
- 引文：引用上级文件时加书名号《》，并注明文号
