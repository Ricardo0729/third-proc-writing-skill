# 文种规则

> 检察机关常用公文文种写作规范。基于《党政机关公文处理工作条例》《人民检察院公文处理办法》等通用规范。

> 📎 另有「用户提供版」（`文种规则-用户提供版.md`）基于本部门实际语料提炼，冲突时优先参考。安全/合规规则（R1-R9）为硬性约束，不可覆盖。

---

## 一、工作汇报

| 项目 | 说明 |
|------|------|
| 用途 | 向院领导、上级院汇报某一时期或专项工作开展情况 |
| 结构 | 背景 → 主要做法 → 取得成效 → 存在问题 → 下一步打算 |
| 语气 | 客观陈述，实事求是 |
| 结语 | "特此汇报"或自然收束，不强制固定结语 |
| 注意 | "成效"部分避免夸大，以数据和事实为依据；"问题"部分要具体，不能回避 |
| 篇幅 | 1500-3000 字（专项汇报）；500-1500 字（常规月报/周报） |

## 二、请示

| 项目 | 说明 |
|------|------|
| 用途 | 请求上级院或本院领导批示、批准事项 |
| 结构 | 背景/依据 → 请示事项 → 拟办建议 → 结语 |
| 语气 | 恳切、简洁、条理清晰 |
| 结语 | "妥否，请批示""以上请示，请予审批" |
| 注意 | **一文一事**；不能多头请示；不能夹带报告事项 |
| 篇幅 | 300-800 字 |

## 三、报告

| 项目 | 说明 |
|------|------|
| 用途 | 向上级院报告工作情况、答复询问、报送材料 |
| 结构 | 依据/背景 → 具体情况 → 结论/建议 |
| 语气 | 客观、全面、严谨 |
| 结语 | "特此报告" |
| 注意 | 报告中**不得夹带请示事项**；如需请示，另拟请示件 |
| 篇幅 | 500-2000 字 |

## 四、情况说明

| 项目 | 说明 |
|------|------|
| 用途 | 就某一具体事项（涉案情况、信访情况等）作出说明 |
| 结构 | 事由 → 基本情况 → 处理情况 → 下一步意见 |
| 语气 | 平实、客观、完整 |
| 结语 | "特此说明"或根据收文对象确定 |
| 注意 | 以事实为基础，区分"已查明事实"和"尚在核查内容" |
| 篇幅 | 300-1500 字 |

## 五、会议纪要

| 项目 | 说明 |
|------|------|
| 用途 | 记载会议议定事项、工作部署 |
| 结构 | 时间地点参会人员 → 会议内容摘要 → 议定事项 |
| 语气 | 客观记录，不评价不渲染 |
| 结语 | 无固定结语，自然收束 |
| 注意 | 只记议定事项，不记讨论过程；"会议指出""会议要求""会议决定"三种层级分明 |
| 篇幅 | 500-2000 字 |

## 六、信息简报（检察信息）

| 项目 | 说明 |
|------|------|
| 用途 | 向上级院或院领导报送工作动态、经验做法、问题建议 |
| 结构 | 导语（核心信息）→ 具体做法/情况 → 简评或效果 |
| 语气 | 简洁、凝练、重点突出 |
| 结语 | 可不设结语，自然收束 |
| 注意 | 标题要提炼核心亮点；导语包含"何人何时何地何事"要素；一事一报 |
| 篇幅 | 500-1000 字 |

## 七、领导讲话稿

| 项目 | 说明 |
|------|------|
| 用途 | 院领导在会议、活动上的讲话 |
| 结构 | 开场（称谓 + 会议背景）→ 主体（认识/回顾/部署）→ 结尾（号召/要求） |
| 语气 | 庄重、有力、有指导性 |
| 结语 | 号召性语句，凝聚共识 |
| 注意 | 需区分"部署性讲话""总结性讲话""致辞"等场景；段落不宜过长；适当使用排比但不滥用 |
| 篇幅 | 2000-5000 字（部署性）；800-2000 字（致辞/简短讲话） |

## 八、函

| 项目 | 说明 |
|------|------|
| 用途 | 与平级单位或不相隶属单位之间商洽工作、询问答复 |
| 结构 | 发函背景 → 具体事项 → 恳请协作事项 → 结语 |
| 语气 | 客气、简洁、平等 |
| 结语 | "特此函告""请予函复""此复" |
| 注意 | 区分"函请"和"函复"，注意称谓 |
| 篇幅 | 200-600 字 |

## 九、案件分析材料

| 项目 | 说明 |
|------|------|
| 用途 | 对某一类案件或具体案件进行综合分析 |
| 结构 | 基本案情 → 争议焦点/特点分析 → 处理意见/建议 |
| 语气 | 审慎、严谨、客观 |
| 结语 | 根据材料性质确定，可不设 |
| 注意 | 不代替承办人作出审查结论；涉及敏感信息必须脱敏 |
| 篇幅 | 1000-3000 字 |
