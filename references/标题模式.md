# 标题模式

> 检察机关常用公文标题规范格式。按文种分类。

> 📎 另有「用户提供版」（`标题模式-用户提供版.md`）基于本部门实际语料提炼，冲突时优先参考。安全/合规规则（R1-R9）为硬性约束，不可覆盖。

---

## 一、工作汇报

### 格式

```
XXXX（部门）关于XXXX（主题）的工作汇报
XXXX（部门）XXXX年X月（季度/半年/年度）工作情况汇报
关于XXXX（专项工作）开展情况的汇报
```

### 示例

```
第三检察部关于2025年上半年工作情况汇报
北海市人民检察院第三检察部关于反洗钱工作情况的汇报
关于打击金融犯罪专项工作进展情况的汇报
```

---

## 二、请示

### 格式

```
关于XXXX（事项）的请示
XXXX（部门）关于XXXX（事项）的请示
```

### 示例

```
关于追加2025年度办案经费预算的请示
第三检察部关于选派人员参加最高检业务培训的请示
关于申请采购电子取证设备的请示
```

```

```

## 六、信息简报

### 格式

**标题应当提炼核心信息，不套用公文式标题。建议使用"做法+效果"结构。**

```
XXXX——北海市检察院第三检察部XXXX工作做法
北海市检察院第三检察部开展XXXX（活动/专项）
XXXX（单位/部门）XXXX（动作）XXXX（效果）
```

### 示例

```
北海市检察院第三检察部强化知识产权检察保护服务创新发展
XX市院第三检察部联合四部门建立反洗钱协作机制
北海市检察院第三检察部开展知识产权保护专题宣传活动
以听证促公正——北海市检察院第三检察部对某某案召开公开听证
```

## 十、案件分析材料

### 格式

```
关于XXXX（类案/专项）的分析报告
XXXX年起XXXX（罪名）案件办理情况分析
XXXX（名称）专题分析报告
XX案相关问题分析
```

### 示例

```
近三年非法吸收公众存款案件办理情况分析报告
2025年电信网络诈骗关联犯罪态势分析
某某案相关法律适用问题分析
```

---

## 注意事项

1. 标题中的部门名称与落款单位一致
2. 标题中不使用"关于+关于"重复结构
3. 标题长度一般不超过40字（含标点）
4. 信息简报标题不套用正式公文格式，突出信息性
5. 同一事项不同文种标题需区分，避免混用
