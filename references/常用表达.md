# 常用表达

> 检察机关公文常用句式与过渡语。按场景分类供调用。

> 📎 另有「用户提供版」（`常用表达-用户提供版.md`）基于本部门实际语料提炼，冲突时优先参考。安全/合规规则（R1-R9）为硬性约束，不可覆盖。

---

## 一、开头用语

### 背景/依据类

| 表达 | 适用场景 |
|------|----------|
| 根据《……》及上级院相关工作要求，…… | 专项工作汇报/方案 |
| 按照市委/上级院统一部署，…… | 贯彻落实性公文 |
| 为贯彻落实……精神，…… | 工作部署/方案 |
| 根据年度工作安排，…… | 常规工作汇报 |
| 受……委托，…… | 报告/讲话稿 |
| 某年某月以来，我院第三检察部围绕……，…… | 阶段性工作汇报 |
| 现将有关情况报告如下： | 各种报告通用过渡句 |
| 现将……情况汇报如下： | 工作汇报过渡句 |

### 时间类

| 表达 |
|------|
| 今年以来 |
| 某年某月至某年某月 |
| 近期 |
| 目前 |
| 截至目前 |

## 二、主体常用表达

### 做法类

| 表达 |
|------|
| 一是……。二是……。三是……。 |
| 围绕……重点开展工作。 |
| 针对……问题，采取以下措施： |
| 联合……开展…… |
| 对……进行梳理排查 |
| 建立健全……机制 |
| 注重……与……相结合 |
| 加强……协作配合 |

### 成效类（审慎使用）

| 表达 | 使用说明 |
|------|----------|
| 共受理审查逮捕案件XX件XX人 | 精确数据 |
| 共提起公诉XX件XX人 | 精确数据 |
| 追赃挽损XX万元 | 精确数据 |
| 发出检察建议XX份，采纳XX份 | 精确数据 |
| 相关指标持续向好 | 有数据支持时使用 |
| 工作取得阶段性进展 | 偏正面但不夸张 |
| 办案质效稳步提升 | 有数据对比时使用 |

### 问题类

| 表达 |
|------|
| 工作中仍存在以下不足： |
| 一是个别案件办理周期较长；二是…… |
| 部分……有待进一步加强 |
| ……衔接配合还不够顺畅 |
| 人员力量与工作任务量之间的矛盾较为突出 |
| 相关制度机制仍需进一步完善 |

### 下一步类

| 表达 |
|------|
| 下一步，将重点抓好以下工作： |
| 持续深化…… |
| 进一步加大……力度 |
| 着力提升……水平 |
| 待条件成熟后，…… |
| 拟于近期开展…… |

## 三、结尾用语

| 文种 | 结语 |
|------|------|
| 工作汇报 | "特此汇报。" / 自然收束 |
| 请示 | "妥否，请批示。" / "以上请示，请予审批。" |
| 报告 | "特此报告。" |
| 情况说明 | "特此说明。" / 自然收束 |
| 函（去函） | "特此函告。" / "请予函复。" |
| 函（复函） | "此复。" |
| 函（商洽） | "以上事项，烦请予以支持。" |
| 通知 | "特此通知。" |
| 信息简报 | 自然收束，不设固定结语 |

---

## 四、衔接与过渡

| 场景 | 表达 |
|------|------|
| 引出问题 | "同时，工作中也存在一些问题和不足：" |
| 引出成效 | "经过共同努力，主要取得以下成效：" |
| 引出分析 | "经梳理分析，主要原因有以下方面：" |
| 引出部署 | "针对上述问题，下一步拟采取以下措施：" |
| 补充信息 | "另，……" |
| 并列关系 | "同时""此外""同步推进" |
| 因果关系 | "因此""据此""鉴于此" |
| 转折关系 | "但""然而""不得不指出的是" |
| 强调 | "特别是""尤其是""需重点关注的是" |

## 五、部门常用术语（第三检察部相关）

| 领域 | 术语 |
|------|------|
| 经济犯罪 | 非法吸收公众存款、集资诈骗、洗钱、虚开增值税专用发票、合同诈骗 |
| 金融犯罪 | 内幕交易、操纵证券市场、违法发放贷款、信用卡诈骗 |
| 知识产权 | 侵犯著作权、假冒注册商标、商业秘密 |
| 专项工作 | 反洗钱、"一案双查"、第三方监督评估 |
| 侦查监督 | 立案监督、侦查活动监督、纠正违法通知书、追捕追诉 |
| 审判监督 | 抗诉、再审检察建议 |
