# 范文 / 脱敏参考样本说明

> 本部门公文写作参考样本管理说明。

---

## 当前状态

**暂无脱敏范文。**

原因：

1. **未获取到本部门的真实公文样本**——无法直接模仿部门实际风格。
2. 如后续提供可公开的脱敏样本，将作为 reference 文件添加到本目录下，按文种分类。
3. 在样本到位之前，写作将依据 references/ 中的通用规范和 templates/ 中的通用模板进行。

## 文件命名规范（如后续添加）

```
【序号】-【文种】-【主题简述】.md
```

例如：

```
001-工作汇报-2025年上半年经济犯罪检察工作汇报.md
002-信息简报-知识产权检察保护做法.md
003-请示-选派人员参加业务培训.md
```

## 使用前提

- 必须为本部门真实公文且已做完整脱敏处理（当事人、案号、金额、地址等全部替换）
- 脱敏后的材料不违反保密规定
- 经部门负责人或内勤审核后方可放入

---

## 替代方案

在范文样本到位之前，写作时将：

1. 依赖 templates/ 中的结构化模板作为骨架
2. 依赖 references/ 中的风格规则、常用表达、标题模式确保用语规范
3. 所有不确定内容标注【待补充】，不虚构
