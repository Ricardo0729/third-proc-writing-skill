# 北海市人民检察院第三检察部公文撰写 Skill

> Claude Code Skill — 起草、修改、润色检察机关内部公文。

## 快速开始

在 Claude Code 中直接说：

```
帮我写一份第三检察部2025年上半年经济犯罪检察工作汇报
```

Skill 会自动激活，先出提纲让你确认，确认后生成正文 + 自查清单。

## 这是什么

一个面向检察机关公文写作场景的 Claude Code Skill，覆盖工作汇报、请示、报告、情况说明、会议纪要、信息简报、领导讲话稿、案件分析材料等常用文种。

核心设计理念：
- **事实为本**：不编造任何事实、数据、法律依据
- **不确定即标注**：缺失内容使用 `【待补充】` 占位
- **先提纲后正文**：降低跑偏概率，减少审阅成本
- **输出三件套**：提纲 + 正式稿 + 自查清单

## 适用文种

| 文种 | 用途 |
|------|------|
| 工作汇报 | 向院领导/上级院汇报工作开展情况 |
| 请示 | 请求批准事项（一文一事） |
| 报告 | 向上级院报告工作/答复询问 |
| 情况说明 | 就某一具体事项作出说明 |
| 会议纪要 | 记载会议议定事项 |
| 信息简报 | 报送工作动态、经验做法 |
| 领导讲话稿 | 部署性/总结性/致辞 |
| 函 | 平级单位商洽工作 |
| 案件分析材料 | 类案或个案综合分析 |
